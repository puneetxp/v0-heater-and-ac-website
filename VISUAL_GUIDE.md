## Responsive Breakpoints

### Desktop (1024px and above)
- 3-column variant grid
- 3-column plan grid
- Side-by-side layout (image + info)
- Table-based plan comparison

### Tablet (768px - 1023px)
- 2-column variant grid
- 2-column plan grid
- Stacked sections with adequate spacing
- Mobile-friendly dropdowns available

### Mobile (Below 768px)
- Single column layouts
- Dropdown selectors instead of grids
- Stacked cards
- Optimized touch targets (44px minimum)
- Collapsed comparison table into accordion
